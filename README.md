# Winline Giveaway — Telegram Mini App

Первая рабочая версия мини-аппа для розыгрышей в фирменном стиле Winline
(оранжевый / графит / белый). Статический фронтенд — деплоится на Vercel
без сборки.

Role-aware SPA в фирменном чёрно-оранжевом стиле Winline (по референсу
winline.ru): курсивные заглавные заголовки, кнопки-пилюли, графитовые карточки.
Статический фронтенд — деплоится на Vercel без сборки.

## Что уже работает

**Навигация:** нижний таб-бар, зависит от роли. Участник видит «Лента / Мои /
Профиль», у админа добавляется вкладка «Админка». Роль для демо переключается в
Профиле (в бою выдаётся по TG-юзеру).

**Участник:**
- **Лента** — карточки активных розыгрышей с живыми таймерами и иконками условий.
- **Карточка розыгрыша** — приз, условия со статусами, кнопка участия.
- **Участие** — авторизация Winline + пошаговая проверка условий (демо-анимация).
- **Мои** — участия со статусом условий и история выигрышей.
- **Профиль** — привязка/отвязка Winline, 18+, переключатель роли.

**Админ:**
- **Дашборд** — свои розыгрыши: идут / черновики / завершённые.
- **Модульный конструктор** — оформление + добавляемые блоки условий (подписка /
  промокод / ставка-депозит; обязательных нет), места, кросспост-каналы, дата.
  Живой предпросмотр. Публикация и черновики.
- **Управление** — пауза/возобновление, досрочное завершение, дублирование,
  редактирование, кросспостинг, счётчик участников.
- **Итоги** — случайные победители + проверочный хеш (provably-fair) + рерол.
- **Аналитика** — KPI, таблица условий 🟢/🔴, выгрузка в Google Sheets.
- **Команда** (супер-админ) — выдача/отзыв доступа по @username.

Интеграция с **Telegram WebApp SDK**: тема, кнопка «Назад», haptic, разворот.

> подписок, Google Sheets, постинг в канал, planscheduler) помечены в `app.v2.js`

## Деплой на Vercel

**Вариант A — через GitHub (рекомендую):**
1. Залей папку `winline-giveaway-app` в репозиторий GitHub.
2. На vercel.com → **Add New → Project** → импортируй репозиторий.
3. Framework Preset: **Other**. Build Command — пусто. Output Directory — пусто
   (корень). Жми **Deploy**.

**Вариант B — через CLI:**
```bash
npm i -g vercel
cd winline-giveaway-app
vercel        # preview
vercel --prod # продакшн
```

Получишь URL вида `https://winline-giveaway.vercel.app`.

## Подключение как Telegram Mini App

1. В Telegram открой **@BotFather** → `/newapp` (или `/mybots` → выбрать бота →
   **Bot Settings → Configure Mini App**).
2. Укажи URL твоего деплоя на Vercel.
3. Открой мини-апп через меню бота или кнопку — он развернётся на весь экран
   и подхватит тему Telegram.
