# Winline Giveaway — Telegram Mini App

Первая рабочая версия мини-аппа для розыгрышей в фирменном стиле Winline
(оранжевый / графит / белый). Статический фронтенд — деплоится на Vercel
без сборки.

## Что уже работает

- **Создание розыгрыша** — визард на 9 шагов (медиа+текст → кнопка → каналы →
  промокод → ставка → категория → места → канал публикации → дата/время),
  живой предпросмотр карточки и сводка условий.
- **Мои розыгрыши** — список с сохранением (localStorage), статусы «идёт / завершён».
- **Итоги** — случайный выбор победителей (стабильный по ID), перекрутка,
  публикация в канал.
- **Аналитика** — KPI, таблица участников со статусами условий 🟢/🔴,
  кнопка выгрузки в Google Sheets.
- **Глазами участника** — публичная карточка + авторизация Winline и
  пошаговая проверка условий (демо-анимация).
- Интеграция с **Telegram WebApp SDK**: тема, кнопка «Назад», haptic-отклик,
  разворот на весь экран.

> Демо-режим: данные хранятся локально в браузере. Реальные интеграции
> (авторизация Winline API, проверка подписок на каналы, запись в Google Sheets,
> отправка постов в канал) помечены в `app.js` комментариями `TODO:BACKEND`.

## Деплой на Vercel

**Вариант A — через GitHub (рекомендую):**
1. Залей папку `winline-giveaway-app` в репозиторий GitHub.
2. На vercel.com → **Add New → Project** → импортируй репозиторий.
3. Framework Preset: **Other**. Build Command — пусто. Output Directory — пусто
   (корень). Жми **Deploy**.

**Вариант B — через CLI:**
```bash
npm i -g vercel
cd winline-giveaway-app
vercel        # preview
vercel --prod # продакшн
```

Получишь URL вида `https://winline-giveaway.vercel.app`.

## Подключение как Telegram Mini App

1. В Telegram открой **@BotFather** → `/newapp` (или `/mybots` → выбрать бота →
   **Bot Settings → Configure Mini App**).
2. Укажи URL твоего деплоя на Vercel.
3. Открой мини-апп через меню бота или кнопку — он развернётся на весь экран
   и подхватит тему Telegram.
